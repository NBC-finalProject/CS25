## 🏆 문제 풀이 서비스

![image](https://github.com/user-attachments/assets/11d2ad4a-2554-4f43-8942-43d2f891cfd8)

- 오늘의 문제
    - 구독자의 정답률에 맞고 중복 없는 오늘의 문제 뽑기
    - 문제의 유형에 따른 점수 차등 제공
        - 객관식/주관식/서술형 문제 유형
        - 쉬움/보통/어려움 난이도
          
- 회원 서비스
    - 회원에게 제공하는 프로필 상세 보기
        - 나의 구독 이력 확인하기
        - 틀린문제 다시 보기
        - 통계로 보는 나의 취약점
          
- AI 채점
    - 서술형 답안 제출시 AI 기반 채점 및 피드백 제공
    - AI가 만들어주는 CS 문제
    - AI가 제공하는 최신 기술 아티클
